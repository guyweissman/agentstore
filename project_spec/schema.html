AgentStore — Database Schema

SQLite metadata store. File content lives in the content-addressed object store (referenced by object_hash), not in any table.

Identity & Access Control Files & heads Commit log PK primary key FK foreign key

Entity relationships

Identity & Access Control

principals
Repo members. Synced snapshot of each member's identity. Grants/roles key on id, never username or key.
id PKTEXT · principal_<uuid>
usernameTEXT · unique per remote
public_keyTEXT · ed25519 (OpenSSH)
created_atINTEGER · ms
repo_roles
Control plane. admin is the only role in v0.1: governs membership and is the data-plane recovery authority.
principal_id FK→ principals.id
roleTEXT · "admin"
granted_by FK→ principals.id
created_atINTEGER · ms
grants
Access control rules. Exact path or prefix match. Current state only; grant/revoke history goes to the server log.
principal_id PK FK→ principals.id
path_pattern PKTEXT · exact or prefix
permissionTEXT · read|write|owner
granted_by FK→ principals.id
created_atINTEGER · ms
PK = (principal_id, path_pattern)

Files & heads

files
Registry of every path that has ever existed in the repo.
path PKTEXT · /strategy/icp.md
created_atINTEGER · ms
deleted_atINTEGER · ms, null if active
file_branch_heads
Most recent accepted commit per file per branch. The OCC unit and fast content lookup. Only main in v0.1.
file_path PK FK→ files.path
branch PKTEXT · "main"
commit_id FK→ commits.id
PK = (file_path, branch)

Commit log

commits
The commit log. Three IDs, one job each: id = content, seq = order/cursor, created_at = wall clock.
id PKTEXT · SHA-256 hex (64)
seqINTEGER · monotonic, server-assigned
messageTEXT
author_id FK→ principals.id
created_atINTEGER · ms
commit_parents
Parent commit(s). Normal: 1 row. First commit: 0 rows. Merge commit: 2 rows. parent_id self-references commits.
commit_id PK FK→ commits.id
parent_id PK FK→ commits.id (self)
orderINTEGER · 0 first, 1 second
PK = (commit_id, parent_id)
commit_files
File changes in each commit. A row is a file version: the (commit, path) pair. object_hash points into the object store.
commit_id PK FK→ commits.id
path PK FK→ files.path
object_hashTEXT · → object store, null if deleted
sizeINTEGER · bytes
change_typeTEXT · added|modified|deleted
PK = (commit_id, path)
